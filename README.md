# Atividade Prática: Comparador de Preços (Dólar vs Real)

Projeto desenvolvido como atividade de presença para a disciplina de Desenvolvimento Mobile.

## 📝 Descrição da Atividade
O objetivo é criar um aplicativo em Flutter que compare o preço de dois produtos similares:
- **Produto 1:** Valor em Dólar (USD).
- **Produto 2:** Valor em Reais (BRL).
- **Cotação:** Campo para inserir o valor atual do Dólar.

**Lógica:** Ao clicar no botão "Comparar", o aplicativo converte o valor do Produto 1 para Reais (multiplicando pela cotação) e informa qual produto está mais barato ou se os preços são iguais.
